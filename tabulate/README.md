# Tabulate

Grocery receipt tracker with OCR + AI categorization, running as a Home Assistant add-on.

Upload receipt photos, let Tesseract + Claude Vision extract items and prices, review and verify, then track your spending trends over time.

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

## Configuration

| Option | Required | Description |
|--------|----------|-------------|
| `anthropic_api_key` | Yes | Anthropic API key for Claude Vision |
